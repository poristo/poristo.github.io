About Project

NTTDATA wanna build a Payment Gateway merchant portal make it possible for merchants to monitor and manage their payment more efficiency. It provides merchants with essential features to facilitate their payment operation. Merchant portal contains transaction dashboard, account and role management, activities history log, settlement and refund.

Project Goals

  • #1 Improve the frontend

  • #2 Design the Dashboard page

  • #3 Create a modern visual style

My duty of work

I thought of it as a good opportunity to test out the user-centered design process. I have explored the same on the payment dashboard ( Ref. Razorpay/Braintree/PayPal/Stripe ).


First of all, the last version dashboard is build by Bootstrap with Jquery. To my knowledge Jquery is out of date ( 2018 ). The future trend js framework should like Vue/ React / Angular. So I have suggested use react & some UI lib ( also discussed with tech lead ).


As I do not have any experience on SAAS, I had to research some payment dashboard to come up with a flow that identified several UX frictions I have faced in the process.
